## 🌟 Présentation du Projet
Cette application permet aux particuliers de trouver des artisans de confiance dans la région Auvergne-Rhône-Alpes. Elle propose une recherche par nom et par catégorie (Bâtiment, Services, Fabrication, Alimentation).

## 📋 Prérequis
Avant de commencer, assurez-vous d'avoir installé :
* **Node.js** (Version 18 ou supérieure)
* **Git**
* **MySQL**
* Un gestionnaire de paquets comme **npm**
